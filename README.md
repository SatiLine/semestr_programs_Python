# Platformer Game (PyQT6)

2D платформер на Python с графическим интерфейсом PyQt6: управление персонажем, враги, монеты, атака мечом, несколько уровней и сохранение статистики в SQLite.

## Возможности
+ Движение, прыжки и гравитация (платформенная физика)​
+ Враги с патрулированием и нанесением урона​
+ Монеты (+10 очков) и очки за врагов (+50 очков)​
+ Атака мечом (ЛКМ) с кулдауном и короткой анимацией​
+ Система жизней и здоровья (индикатор в интерфейсе)​
+ Таблица рекордов (топ-10) и детальная статистика игрока в SQLite​
+ Пауза, выход в меню с предложением сохранить прогресс

## Установка и запуск
Вариант 1 — запуск из исходников
1. Установите Python 3.8+ (https://www.python.org)
2. Установите зависимости:
`pip install PyQt6`
3. Скачайте/распакуйте проект в любую папку.
4. Запустите приложение:
`python main.py`

## Начало работы
1. После запуска откроется главное меню.​
2. Нажмите «Новая игра».​
3. Введите имя игрока (или оставьте по умолчанию «Гость») и нажмите OK.​
4. Чтобы посмотреть рекорды: в меню нажмите «Таблица рекордов» и закройте окно кнопкой «Закрыть».

## Управление
Клавиша -- Действие
A -- движение влево
D -- Движение вправо 
Space -- Прыжок (только если персонаж стоит на платформе)
ЛКМ -- Атака мечом
Enter -- Пауза / продолжение
Esc -- Выход в меню (с запросом сохранения)

## Особенности:
1. Персонаж останавливается автоматически, если отпустить A/D.​
2. Прыжок фиксированный, не зависит от длительности нажатия Space.​
3. Атака мечом имеет перезарядку ​
4. Направление атаки зависит от направления движения: при удержании A меч появляется слева, иначе — справа.

## Игровые механики
#### Объекты
Игрок: здоровье 100, жизней 3; цвет меняется в зависимости от здоровья и атаки.​
Монеты: дают +10 очков, исчезают при подборе.​
Враги: при касании наносят 10 урона, при убийстве дают +50 очков.​
Платформы: используются для перемещения и прыжков; при ударе снизу прыжок останавливается.​

### Как пройти уровень
Чтобы уровень засчитался, нужно:
собрать все монеты на уровне;
победить всех врагов на уровне.​

После этого появляется сообщение о прохождении и автоматически загружается следующий уровень, здоровье восстанавливается до 100, позиция игрока сбрасывается на старт.

### Смерть и Game Over
Если здоровье стало 0: уровень перезапускается и уменьшается число жизней, увеличивается счётчик смертей в статистике.​

Если жизни закончились: игра завершится, результат сохранится в базу, затем откроется главное меню.

### Очки и статистика
Начисление очков:
Монета: +10​
Враг: +50

Сохраняется в SQLite:
имя игрока, итоговый счёт, достигнутый уровень, время игры;
общее число монет, убийств, смертей;
статистика по уровням (количество прохождений, лучшее время).

Таблица рекордов
В окне рекордов показывается топ-10 результатов, отсортированных по:
наибольшему счёту;
при равенстве — по меньшему времени.​
Колонки: место, имя, счёт, уровень, время, монеты, убийства, смерти.

Пауза и выход
Enter ставит игру на паузу и повторное нажатие возобновляет игру.​
Esc открывает диалог выхода в меню: можно сохранить прогресс (Yes), выйти без сохранения (No) или отменить (Cancel).
